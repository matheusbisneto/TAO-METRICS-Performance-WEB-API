2. Passos para Rodar o Projeto no VS Code (ou terminal comum)
2.1. Abra o terminal do VS Code (ou terminal comum)
Abra o terminal dentro do VS Code ou qualquer terminal no seu computador.

Navegue até a pasta do projeto:

cd C:\Users\User\Documents\UNIBTA\2025PM

2.2. Ative o Ambiente Virtual
No Windows, ative o ambiente virtual executando o seguinte comando no terminal:
venv\Scripts\activate
Você verá algo como (venv) no início da linha do terminal, indicando que o ambiente virtual foi ativado.

2.3. Execute o Streamlit
Agora, execute o Streamlit normalmente com o seguinte comando:

streamlit run taometrics.py
Ou, se o seu sistema precisar de aspas:
streamlit run "taometrics.py"

Isso abrirá o aplicativo no navegador. Agora você pode acessar a aplicação localmente no navegador!

3. Rodar o Projeto em Outro PC
Caso você queira rodar o projeto em outro computador, siga as etapas abaixo:

3.1. Entre na Pasta do Projeto
No novo computador, entre na pasta do projeto:

cd C:\Users\User\Documents\UNIBTA\2025PM

3.2. Crie um Novo Ambiente Virtual
Crie um novo ambiente virtual executando o comando:

python -m venv venv

3.3. Ative o Ambiente Virtual
Ative o ambiente virtual com o seguinte comando:

No Windows:

venv\Scripts\activate

3.4. Instale as Dependências
Instale todas as bibliotecas necessárias (aquelas que seu projeto precisa para rodar) executando:

pip install -r requirements.txt
Note: Se você não tiver o arquivo requirements.txt (caso você tenha enviado a pasta venv com todas as dependências), pode ser necessário criar esse arquivo. Você pode gerar esse arquivo com o comando:

pip freeze > requirements.txt
3.5. Execute o Streamlit
Agora, execute o Streamlit normalmente com o seguinte comando:

streamlit run taometrics.py
Isso abrirá o aplicativo no navegador do novo computador!
